How to use the PCMA online application ?

Visit: /PCMA/predictor/
  • First : Choose Analysis model between four options.

    • PCMA analyzes the pathways of ‘single bacterial taxon - PC of multiple metabolites - disease’.

    PCMA2 collects and performs PCA on all the metabolites correlated with the disease at one time, followed by MA on each ‘bacterial taxon - PC - disease’ pathway.

    PCPCMA performs PCA on multiple bacterial taxa, before mediation analysis along the ‘PC of bacterial taxa - PC of metabolites - disease’ pathways.

    Attention: the first column of the three files (record numbers of patients) must be same.
  • Upload the Bacteria_file, Metabolite_file and Diagnosis_file.

    • Bacteria relative abundance matrix (.csv table files): Bacteria file contains bacteria relative abundance for multiple samples. The first column shows record numbers of patients. The first row reveals bacteria species.

    Metabolites concentration matrix (.csv table files): Metabolite file contains metabolites concentrations for multiple samples. The first column shows record numbers of patients. The first row should be bacteria species. The concentrations data are unstandardized in default. Parameter should be specified to refuse standardization for standardized data.

    Disease phenotype matrix (.csv table files): Diagnosis file contains disease phenotypes for multiple samples. There are two columns including record numbers of patients and binary outcomes which stand for disease phenotypes or diagnoses (0 or 1 variable). Users must convert disease phenotypes, control group and disease group for instance, into binary variables. Suggest you set the control group to 0 and the disease group to 1. The first row should be header.

    Functional annotation table (.csv table files): Users should offer a functional annotation table in case the function analysis is expected. The label file must be a comma-separated values file including metabolomic feature in the first column and functional annotation in the second column, two columns in total. The first row should be header.


  • Offer an email address when you submit your job so that you will be informed once your job is finished.

  • An unique jobID will be generated once you submit your job successfully. You can check your jobID via your email and get your results by using the ID in /PCMA/result.

  • Standalone Version of PCMA is freely available at /PCMA/download.html.

How to explain the results you got from the PCMA website?

  • The program will generate a folder named result, and all the results will appear in the folder. The result folder goes down to two folders named data and plot.

    • The folder data includes all data generated from PCMA analysis in .csv format. The paramters_list.csv file contains all the entered parameters. The folder plot contains all the generated images. The images are composed of vector pdf files and 600DPI png images of the same name, including heat map, volcano map, box plot, etc. In addition, a PDF report is automatically generated under the result folder, which explains the contents of the data and plot folder.

  • The program will generate a PDF report contains parameters table, volcano map, Sankey plot, box plot, etc., indicating significant features and PCs after PCMA analysis.

    • The volcano map reveals the selection criteria of mediator variable according to metabolite or other intermediate. The heatmaps visualizes the changes in mediator variable before and after normalization. The PCA Metabolite Heatmap reflects the principal components (PCs) derived from the mediator variable after PCA processing, where each PC is linearly composed of the original variables with certain coefficients. Sankey plot indicates correlation among bacteria, metabolites, and disease phenotypes in order. Boxplots are used to show significant Bacteria and corresponding PCs, as well as Metabolite and corresponding PCs.
